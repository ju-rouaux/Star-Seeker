{\footnotesize\ttfamily DECLSPEC char$\ast$ SDLCALL SDL\+\_\+\+Get\+Clipboard\+Text (\begin{DoxyParamCaption}\item[{void}]{ }\end{DoxyParamCaption})}

Get UTF-\/8 text from the clipboard, which must be freed with SDL\+\_\+free().

This functions returns empty string if there was not enough memory left for a copy of the clipboard\textquotesingle{}s content.

\begin{DoxyReturn}{Renvoie}
the clipboard text on success or an empty string on failure; call \mbox{\hyperlink{SDL__error_8h_ac88e1aa404a8f94aaee4ad6b94547647}{SDL\+\_\+\+Get\+Error()}} for more information. Caller must call SDL\+\_\+free() on the returned pointer when done with it (even if there was an error).
\end{DoxyReturn}
\begin{DoxySince}{Depuis}
This function is available since SDL 2.\+0.\+0.
