\doxysubsection{Description détaillée}
Include file for SDL game controller event handling

In order to use these functions, \mbox{\hyperlink{SDL_8h_a8fc8d35348d7c74bad8392d776c937b8}{SDL\+\_\+\+Init()}} must have been called with the \mbox{\hyperlink{SDL_8h_ac224a9aa4208de0e9b03948ff9b5fe0a}{SDL\+\_\+\+INIT\+\_\+\+GAMECONTROLLER}} flag. This causes SDL to scan the system for game controllers, and load appropriate drivers.
