{\footnotesize\ttfamily DECLSPEC void SDLCALL SDL\+\_\+\+Set\+Main\+Ready (\begin{DoxyParamCaption}\item[{void}]{ }\end{DoxyParamCaption})}

Circumvent failure of \mbox{\hyperlink{SDL_8h_a8fc8d35348d7c74bad8392d776c937b8}{SDL\+\_\+\+Init()}} when not using SDL\+\_\+main() as an entry point.

This function is defined in \mbox{\hyperlink{SDL__main_8h}{SDL\+\_\+main.\+h}}, along with the preprocessor rule to redefine main() as SDL\+\_\+main(). Thus to ensure that your main() function will not be changed it is necessary to define SDL\+\_\+\+MAIN\+\_\+\+HANDLED before including \mbox{\hyperlink{SDL_8h}{SDL.\+h}}.
