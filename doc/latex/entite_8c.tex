\doxysubsection{Description détaillée}
Module de manipulation des entites. 

Pour faciliter le développement, ce module permet l\textquotesingle{}affichage, l\textquotesingle{}animation et le déplacement automatique d\textquotesingle{}une entité selon ses paramètres.
