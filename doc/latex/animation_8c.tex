\doxysubsection{Description détaillée}
Module facilitant la gestion des animations. 

Ce module utilise les configurations données dans la structure \mbox{\hyperlink{structt__animation}{t\+\_\+animation}} pour se charger d\textquotesingle{}animer une texture avec update\+Animation.
