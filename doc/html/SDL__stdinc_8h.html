◆ SDL_acos()

DECLSPEC double SDLCALL SDL_acos ( double  x)

Use this function to compute arc cosine of x.

The definition of y = acos(x) is x = cos(y).

Domain: -1 <= x <= 1

Range: 0 <= y <= Pi

Paramètres
xfloating point value, in radians.
Renvoie
arc cosine of x.
Depuis
This function is available since SDL 2.0.2.
