Calcule l'animation à joueur selon l'orientation du joueur.

Les codes pour code_animation sont les suivants : -0 : innactif -1 : marche -2 : attaque épée -3 : projectile

code_animationSpécification de l'animation désirée

L'identifiant de l'animation à jouer.

Méthode appelée à chaque frame, actualisant le joueur.

Permet de redéfinir les champs de la structure du joueur selon le contexte de la partie. Cette fonction ne dessine pas le joueur.

Paramètres
moteurLe moteur du jeu
joueurLe joueur
Renvoie
Un code indiquant un éventuel état du joueur, uniquement 0 pour l'instant.
