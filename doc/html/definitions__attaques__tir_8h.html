Description détaillée

Fichier de définition des attaques de tir.

POUR CREER UNE NOUVELLE ATTAQUE :

  • Ajouter le nom de l'attaque dans l'enum e_nom_attaque sous la forme NOMATTAQUE dans le fichier attaque.h
  • Créer une fonction sous la forme : static void attaque_tir_NOMATTAQUE(t_attaque_tir * attaque) { attaque->type_projectile = NOM_DU_TYPE_PROJECTILE; attaque->cooldown = INT; attaque->nb_salves = INT; attaque->nb_proj_salve = INT; attaque->tir_interval = INT; attaque->etalement = FLOAT; }

Ajouter la fonction au switch de chargerAttaqueTir : case NOMATTAQUE: attaque_tir_NOMATTAQUE(attaque); break;
